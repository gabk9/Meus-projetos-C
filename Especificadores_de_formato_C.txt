Printf

Digito            Descrição                                                             Especificador e tipo de dado


d ou i            Números inteiros em base decimal                                      int, short int, long int
-----------------------------------------------------------------------------------------------------------------------------------------
X                 Números inteiros em base hexadecimal                                  unsigned int, unsigned long
-----------------------------------------------------------------------------------------------------------------------------------------
f                 Números em ponto flutuante (com casas decimais)                       float
-----------------------------------------------------------------------------------------------------------------------------------------
lf                Números em ponto flutuante (com muito mais casas decimais)            double
-----------------------------------------------------------------------------------------------------------------------------------------
e                 Números em notação científica (com casas decimais)                    float, double
-----------------------------------------------------------------------------------------------------------------------------------------
p                 Usado para imprimir o endereço de uma variável ou um ponteiro         void *, int *, char *, etc.
-----------------------------------------------------------------------------------------------------------------------------------------
c                 Caracteres alfanuméricos (única letra)                                char
-----------------------------------------------------------------------------------------------------------------------------------------
s                 Sequência de caracteres alfanuméricos (sequência de letras)           char[]
-----------------------------------------------------------------------------------------------------------------------------------------
.<num>            Especifica quantos dígitos serão impressos após a vírgula             Usado com float, double
-----------------------------------------------------------------------------------------------------------------------------------------
zu                Para valores do tipo size_t                                           size_t
-----------------------------------------------------------------------------------------------------------------------------------------
u (com cast)      Para imprimir unsigned char como número (0 a 255)                     unsigned char (cast para unsigned int)
