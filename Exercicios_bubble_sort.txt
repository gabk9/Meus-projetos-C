Lista de 10 Exercícios de Bubble Sort em C

1. Ordenar vetor crescente (inteiros)
Objetivo: Ler 10 números e ordená-los em ordem crescente.
Dica: Implemente o Bubble Sort básico.
Exemplo:
int v[10] = {5, 3, 8, 1, 2, 7, 4, 10, 6, 9};



2. Ordenar vetor decrescente
Objetivo: Ler 10 números e ordenar do maior para o menor.
Dica: Troque o sinal > por < na comparação.
Exemplo:
if (v[j] < v[j+1]) // Para ordem decrescente



3. Contar quantas trocas o algoritmo fez
Objetivo: Mostrar quantas trocas foram feitas durante a ordenação.
Dica: Use um contador dentro do if de troca.
Exemplo:
int trocas = 0;
if (v[j] > v[j+1]) { trocas++; }



4. Otimizar o Bubble Sort com flag de parada
Objetivo: Parar o algoritmo se nenhuma troca for feita em uma passada.
Dica: Use uma variável trocou e break se trocou == 0.



5. Ordenar números reais (float)
Objetivo: Usar Bubble Sort com float ao invés de int.
Dica: Apenas mude o tipo dos elementos do vetor.



6. Ordenar apenas os números pares
Objetivo: Ordenar apenas os números pares de um vetor.
Dica: Crie um vetor auxiliar só com os pares, ordene e depois reinserir.



7. Ordenar strings (nomes) em ordem alfabética
Objetivo: Ler 5 nomes e ordenar em ordem alfabética.
Dica: Use strcmp() e strcpy() para comparar e trocar.
Exemplo:
if (strcmp(nomes[j], nomes[j+1]) > 0) { troca... }



8. Ordenar matriz linha por linha
Objetivo: Ordenar cada linha de uma matriz 3x3 separadamente.
Dica: Aplique Bubble Sort em cada linha da matriz com um `for`.



9. Verificar se vetor já está ordenado
Objetivo: Exibir mensagem se o vetor já estiver ordenado antes de aplicar o sort.
Dica: Faça uma verificação simples antes do Bubble Sort.



10. Ordenar vetor e buscar elemento com busca binária
Objetivo: Ordenar vetor e depois buscar um número com binary search.
Dica: Bubble Sort primeiro, depois implementar busca binária.
